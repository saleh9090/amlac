# Workflows

## Building Setup Workflow

1. Create building.
2. Add units to the building.
3. Define unit type.
4. Add layout where applicable.
5. Set unit status as vacant.

## Tenant Workflow

1. Create tenant.
2. Select tenant type: Individual or Company.
3. Add ID/CR.
4. Add contact information.

## Contract Workflow

1. Select tenant.
2. Select unit.
3. Enter contract number.
4. Enter start date.
5. Enter duration in months.
6. System calculates end date automatically.
7. Enter rent amount.
8. Select payment frequency.
9. Select payment method.
10. Save contract as draft or active.
11. When active, system generates contract payments.

## Rent Payments Workflow

1. Open the Rent Payments page.
2. Select building, month, and year.
3. For each unit, enter amount paid, payment date, and note.
4. System updates payment status per unit (Unpaid, Partial, or Paid) based on the unit's contract rent.
5. Totals for the selected building/month/year (total due, total paid, total outstanding) are shown below the table.

## Expense Workflow

1. Select building.
2. Enter expense date.
3. Select expense category (from the manageable Expense Categories list).
4. Enter amount.
5. Enter description.
6. Save expense.

## Expense Category Workflow

1. Open the Expense Categories page.
2. Enter category name.
3. Save.

## Income Workflow

1. Enter income date.
2. Enter amount.
3. Select income category (from the manageable Income Categories list).
4. Enter note.
5. Save income.

## Income Category Workflow

1. Open the Income Categories page.
2. Enter category name.
3. Save.
